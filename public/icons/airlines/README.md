# Airline logos for Commute Assist

Add PNG files here for airline logos. Use the 2-letter IATA code as the filename (e.g. `B6.png`, `UA.png`).

**Recommended:** Square images, 64–128px, transparent or white background.

**Priority list** (most common in commute routes):
- AA (American), B6 (JetBlue), DL (Delta), F9 (Frontier), NK (Spirit), UA (United), WN (Southwest)
- AS (Alaska), G4 (Allegiant), HA (Hawaiian), OO (SkyWest), SY (Sun Country), YX (Republic), MQ (Envoy), 9E (Endeavor)

If a logo is missing, the app falls back to the airline's website favicon.
